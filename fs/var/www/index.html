Zynq-7000 EPP Linux Solution

Downloads

  • Zynq-7000 EPP Linux Image & QEMU Emulator
  • Mentor Sourcery CodeBench GNU toolchain

Developer Resources

  • ISE® Design Suite Embedded Edition
  • Zynq-7000 EPP Extensible Virtual Platform

The Zynq™-7000 EPP Linux solution combines the benefits of the Linux open source operating system together with a customized solution geared towards designing software applications on this platform.

This solution provides:

  • Fully tested and user ready, out-of-the-box solution targeting Zynq-7000 EPP boards
  • Board support package and drivers that are merged into mainline kernel.org tree, providing the benefits of latest kernel features
  • Linux kernel that use device tree for run-time configuration which allows the same kernel image to run on different hardware configurations
  • QEMU system model of the Zynq processing system to run and debug software applications
  • Access to a growing community of software developers interested in the Zynq-7000 EPP platform
  • Access to world-class commercial solutions from Linux partners

  • Mentor Sourcery CodeBench toolchain
